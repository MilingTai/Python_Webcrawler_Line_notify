# Python_Webcrawler_Line_notify
專案介紹
-
本專案以「Farfetch」國外購物網站為例，來示範開發Python網頁爬蟲時，如何爬取所需的資料內容，並連結LINE Notify 傳送訊息到LINE。

前置作業
-
可運用pipenv 或 poetry 套件管理工具
